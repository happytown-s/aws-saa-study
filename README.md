# AWS SAA Study - Free Practice Exam App

A free, open-source study app for the AWS Certified Solutions Architect Associate (SAA-C03) exam. Practice with 300 exam-style questions across all 4 exam domains.

## Features

- 300 exam-style practice questions
- 10 categories covering all SAA-C03 domains
- 3 study modes: Drill (10 Qs), Practice Exam (65 Qs), Review (Wrong Answers)
- Bilingual EN/JA interface with one-click switching
- Hint system for quick reference
- Progress tracking with category performance breakdown
- SAA-C03 score estimation (720/1000 passing threshold)
- Dark theme, mobile-responsive
- No registration required -- 100% free

## Exam Domains Covered

1. Design Resilient Architectures (30%)
2. Design High-Performing Architectures (28%)
3. Design Secure Architectures (24%)
4. Design Cost-Optimized Architectures (18%)

## Categories

- Design Resilient Architectures (30 Qs)
- Design High-Performing Architectures (30 Qs)
- Design Secure Architectures (30 Qs)
- Design Cost-Optimized Architectures (30 Qs)
- EC2 & Compute (30 Qs)
- S3 & Storage (30 Qs)
- VPC & Networking (30 Qs)
- RDS & Databases (30 Qs)
- IAM & Security (30 Qs)
- Monitoring & Logging (30 Qs)

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS

## How to Use

Visit the live app or run locally:

```bash
git clone https://github.com/happytown-s/aws-saa-study.git
cd aws-saa-study
npm install
npm run dev
```
